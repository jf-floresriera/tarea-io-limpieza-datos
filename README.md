# Tarea 10 – Lectura Robusta de Archivos y Limpieza de Datos

## Descripcion

Ejercicios 1 y 2 implementados en Python, R y Julia con documento Quarto.

## Estructura



## Como ejecutar

```bash
python scripts/analisis.py   # Python
Rscript scripts/analisis.R   # R
julia scripts/analisis.jl    # Julia
```

## Renderizar Quarto

```bash
quarto render analisis.qmd --to html --output-dir renders
quarto render analisis.qmd --to pdf  --output-dir renders
```

## eL pdf y HTML renderizado esta en la carpeta:
